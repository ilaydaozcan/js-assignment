# Super Cool Image Carousel

## Description
An image carousel built entirely with HTML, CSS, and JavaScript.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Animations)
- JavaScript

## How to Run
1. Clone this repository to your machine.
2. Ensure `index.html`, `style.css`, and `script.js` are in the same folder.
3. Double-click the `index.html` file to open it in your default web browser.

## Features
* **Smooth Transitions:** Uses CSS transform and visibility properties to create a polished "slide and pop" effect when changing images.
* **Keyboard Support:** Users can seamlessly navigate through the images using the `ArrowLeft` and `ArrowRight` keys on the keyboard.
* **Dynamic Counter:** Automatically calculates the total number of images and displays the user's current position (e.g., "1 of 8").
